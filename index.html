Below are the interactive maps showing the model estimation and simulation results:

  • Map 1: Estimation of Activity Utilities – A detailed visualization of estimated activity utilities across different regions, hours, and user groups. Use the drop-down menu to explore.
  • Map 2: Hourly Parking Activities with User Heterogeneity – Demonstrates hourly parking activities, highlighting variations based on user heterogeneity. Use both the slider and drop-down menu to explore.
  • Map 3: Parking Accuracy for Each Group – Also represented by the mean absolute percentage error (MAPE). Use the slider to explore the accuracy across different hours.
